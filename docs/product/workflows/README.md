# Workflow Templates (v1)

These templates are starter definitions for `workflow_execute`.

## Included
- `client-reporting.json`
- `client-approval-gate.json`
- `client-handoff.json`

## Usage
1. Copy a template.
2. Replace placeholder values (URLs, contacts, content schemas).
3. Run with `workflow_execute`.
4. Track outcomes via `agency_roi_snapshot` or `a2a://agency-roi`.
