# Commercial Offer (v1)

## Offer Name
Agency Cloud Pilot

## Package Includes
- Managed onboarding
- Managed cloud runtime
- Three packaged workflows (reporting, approval gate, handoff)
- Weekly optimization and KPI review
- Priority support channel

## Commercial Structure
- Monthly fee: EUR 1.5k-EUR 3k
- One-time setup fee: required
- Pilot duration: 6 weeks

## Pilot Success Criteria
1. A non-technical ops lead launches one template workflow.
2. Every run has traceability (who, what, when, result).
3. Workspace permissions prevent cross-client data access.
4. KPI baseline and post-pilot delta are measured.

## Expansion Trigger
Offer expansion after pilot only when at least one KPI shows measurable improvement (time saved, throughput, error-rate reduction).
