# Workspace Manager & Prompt Enhancer App

**Concept:** A minimalist, high-productivity dashboard for managing UI design projects and optimizing AI prompts using the Stitch Effective Prompting Guide.

## Structural Foundation
*   **The Dashboard:** A centralized "Command Center" where users can view all their active Stitch projects as distinct, elegant cards. Each card should show a thumbnail of the latest screen, project title, and a status badge.
*   **The Prompt Enhancer:** A dedicated, focused workspace with a split-pane layout.
    *   **Left Pane:** A clean text area for entering raw design intents.
    *   **Right Pane:** A real-time "Enhanced Preview" that applies structural templates (The Detailed Foundation, The Targeted Edit) and suggests UI/UX keywords.
*   **The Screen Gallery:** A masonry-style layout for browsing screens within a project, featuring quick-action buttons for "Edit" and "Generate Variant."

## Stylistic Advice
*   **Vibe:** Professional, focused, and "industrial minimalist."
*   **Color Palette:** A sophisticated dark mode using deep charcoals and muted slate grays, with vibrant "electric blue" accents for active states and primary CTAs.
*   **Typography:** A crisp, high-readability sans-serif font family (e.g., Inter or SF Pro) for UI elements, and a clean monospaced font for prompt code blocks.
*   **Borders & Radii:** Use 8px rounded corners for cards and input fields to maintain a modern, friendly but professional feel. 1px subtle borders (20% opacity) for container separation.

## Iterative Refinement Tasks
1.  On the **Project Card**, add a "Refresh" icon in the top-right corner to manually sync project state.
2.  In the **Prompt Enhancer**, include a "Stitch Guide" toggle that overlays helpful structural tips based on the current text selection.
3.  Ensure all imagery (icons and placeholders) matches the electric blue accent theme for visual harmony.
